# mini program misc

Miscellaneous knowledge of miniProgram! 

## strack of miniProgram :
```
- pages       # 项目基本页集中区
  |- index      # index模块
     |- index.js     # 页用js     
     |- index.json   # 页用json
     |- index.wxml   # 页面 weixin markup language
     |- index.wxss   # 页用样式 weixin style sheets
- utils       # 通用js配置
  |- utils.js     # 实用函数文件
- app.js      # 启动任务与全局数据设置
- app.json    # 项目结构配置与基本样式配置
- app.wxss    # 类 css样式文件 weixin style sheets
```
